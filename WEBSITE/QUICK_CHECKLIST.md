## Emergency Contacts

### If Site Breaks
1. Restore from UpdraftPlus backup
2. Deactivate last changed plugin
3. Contact hosting support
4. Check error logs in hosting panel

### If Hacked
1. Run Wordfence scan
2. Change all passwords
3. Restore clean backup
4. Contact hosting support
5. Check for unauthorized admin users

### Performance Issues
1. Clear all caches
2. Deactivate plugins one by one
3. Test with default theme
4. Check hosting resources
5. Contact hosting support
